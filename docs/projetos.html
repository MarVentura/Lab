Banco P



O projeto Banco P consiste numa plataforma que envolve a possibilidade de ter todas as suas contas num banco apenas, pode também usar saldo de qualquer conta a partir do cartão do BancoP.
Com uma das funcionalidades mais inovadoras do Banco P pode enviar dinheiro para os seus amigos pelo chat.
